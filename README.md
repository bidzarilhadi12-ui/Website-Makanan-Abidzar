# Website-Makanan-Abidzar
Abidzar adalah website listing menu makanan dan minuman yang dirancang untuk membantu pelanggan menjelajahi dan memilih menu dengan mudah. Website ini menampilkan berbagai pilihan menu dalam tampilan grid card yang menarik, dengan informasi lengkap seperti gambar, nama, deskripsi, harga, dan kategori setiap menu. Pengguna dapat mencari menu
